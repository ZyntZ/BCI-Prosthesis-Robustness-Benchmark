# Files recommended for removal from the public repository

These files are generated or duplicated and are not needed to reproduce the numerical results.

## Remove

- `artifacts/`: generated copies of reports, validation output and release manifests. The corresponding commands recreate this directory.
- Any `__pycache__/` directory and `*.pyc` file: Python bytecode.
- `reports/submission_readiness_checks.csv` and `reports/submission_readiness_summary.json`: regenerated by `make submission-readiness`.
- `reports/release_manifest.json`: regenerated by `make release-manifest`.
- `reports/PhysionetMI_PhysionetMI_all_riemann_lr_methods_figures_manifest.json`: internal figure bookkeeping.
- `reports/*_statistical_report_summary.md`: generated Markdown copies of tables already available as CSV/LaTeX. Keep them only if the target journal or users need Markdown output.

## Do not remove

Keep `results/*_results.csv`, `results/*_subject_summary.csv`, the paired-comparison CSV files, source code, tests, configuration, lock files, provenance documentation, manuscript source and manuscript figures. These files support verification or reproduction of the reported results.

Before deleting generated reports, confirm that no manuscript command includes them directly. The current `manuscript/manuscript.tex` uses its own tables and figures.
